# Templates

Starter scaffolds. Copy, don't edit the originals.

- `daily.md` — daily dev log
- `adr.md` — architecture decision record
- `runbook.md` — operational runbook
- `feature-moc.md` — starter for a new feature domain MOC

## For agents

When creating a new doc of a given type, copy the template body (skip the frontmatter `template:` key) and fill in placeholders. Don't leave `{{placeholder}}` values.
