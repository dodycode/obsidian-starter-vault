# Architecture

> High-level architecture docs and diagrams. For code-level conventions, see the app repo's per-package `CLAUDE.md` files.

## What lives here

- System-wide architecture docs (request flow, data model, tenant isolation)
- Excalidraw diagrams (`*.excalidraw.md`) — edit in Obsidian with the Excalidraw plugin
- Migration plans
- Infrastructure overviews

## What does NOT live here

- Per-feature architecture → `Features/<Domain>/`
- Per-app specifics → app repo's `apps/<app>/CLAUDE.md` (or wherever the app lives)
- Per-package internals → app repo's `internal/<pkg>/CLAUDE.md`
